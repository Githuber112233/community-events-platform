# 2026-03-29

## 前端迁移：React → Vue 3

将社区活动平台前端从 React 18 完整迁移到 Vue 3，保持原有 Tailwind CSS 样式不变。

### 主要变更
- package.json：移除 react/react-dom/react-router-dom/lucide-react，添加 vue/vue-router/lucide-vue-next/@vitejs/plugin-vue
- vite.config.ts：从 @vitejs/plugin-react 改为 @vitejs/plugin-vue
- tsconfig.json：添加 .vue 文件支持
- tailwind.config.js：content 增加 `*.vue` 文件扫描
- index.html：挂载点 #root → #app，入口 main.tsx → main.ts
- 所有 .tsx 组件转换为 .vue 单文件组件（SFC）
- React hooks → Vue Composition API（useState→ref/reactive，useMemo→computed）
- React Router → Vue Router（Link→RouterLink，useNavigate→useRouter，useParams→useRoute）

- 图标库改用 lucide-vue-next
